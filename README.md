# MIS3313_GroupProject
Project Overview

This project is a website for UHD Cafe. The goal was to create a clean and simple website that represents a coffee shop where students can study, relax, and enjoy coffee. The website focuses on easy navigation, a consistent design, and a user-friendly layout.

Pages Included
Home Page (Home.html): Introduces the cafe and shows featured drinks.
About Page (about.html): Explains what the cafe offers, such as quality coffee, a calm environment, and customer service.
Services Page (services.html): Displays the menu and services offered by the cafe.
Contact Page (contact.html): Includes a form for users to send messages, feedback, or questions.
Features
Navigation bar on all pages
Responsive design using Bootstrap
Image-based layout to match the cafe theme
Contact form with multiple input fields
Consistent color theme based on UHD colors
Technologies Used
HTML
CSS
Bootstrap
JavaScript
Folder Structure
css/ – contains styles.css
js/ – contains script.js
images/ – contains all images used on the website
HTML files for each page
How to Run
Download the project files
Open Home.html in a web browser
Navigate through the pages using the menu
